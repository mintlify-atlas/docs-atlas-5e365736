## What is Soprano-Factory?

**Soprano-Factory** is a 600-line training script that allows you to train and fine-tune Soprano models with your own data, on your own hardware. You can use Soprano-Factory to add new voices, styles, and languages to the original Soprano model.

Whether you're building custom voice models for production applications or experimenting with new languages and speaking styles, Soprano-Factory provides a straightforward path from dataset to deployed model.

## About Soprano

**[Soprano](https://github.com/ekwek1/soprano)** is an ultra-lightweight, on-device text-to-speech (TTS) model designed for expressive, high-fidelity speech synthesis at unprecedented speed.

<Note>
Soprano-Factory trains models that are fully compatible with the main Soprano inference engine. Once you've trained a custom model, you can use it directly in the [Soprano repository](https://github.com/ekwek1/soprano) by passing your save directory to the `model_path` parameter.
</Note>

### Soprano key features

- Up to **20x** real-time generation on CPU and **2000x** real-time on GPU
- **Lossless streaming** with **&lt;250 ms** latency on CPU, **&lt;15 ms** on GPU
- **&lt;1 GB** memory usage with a compact 80M parameter architecture
- **Infinite generation length** with automatic text splitting
- Highly expressive, crystal clear audio generation at **32kHz**
- Widespread support for CUDA, CPU, and MPS devices on Windows, Linux, and Mac
- Supports WebUI, CLI, and OpenAI-compatible endpoint for production-ready inference
